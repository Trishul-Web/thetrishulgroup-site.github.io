Privacy Policy

Effective Date: May 31, 2026

Devduos ("we", "us", or "our") operates the Rail Junction: Station Empire mobile game application (the "App"). This page details our practices for compiling, collecting, and securing telemetry or usage rules when interacting with our software systems.

1. Information We Collect

Rail Junction: Station Empire values clean player anonymity. Core platform loops operate entirely account-free. We do not require active authorization profiles or intentionally request Personally Identifiable Information (PII) such as customer email lines, names, or addresses.

2. Architecture & Database Storage

All active simulation data, including custom timetable configurations, unlocked engine platforms, layout expansions, and offline currency vaults, are saved locally onto your system hardware via high-speed sandboxed data models. This profile context is never pushed to remote servers.

3. Asset Retention and System Deletion

Data Retention: Game assets, profile values, and achievement progress persist uniquely inside your device's architecture storage for the duration of the software's deployment lifecycle.

Data Deletion: You retain complete mastery over your tracking vectors. If you desire a permanent clear out of local data records, you can execute either option:

  • Application Uninstallation: Deleting the game from your operational unit purges all dependent local memory segments automatically.
  • System Clearing: Run clear loops directly via native configuration paths (Settings > Apps > Rail Junction > Storage > Clear Storage).

4. Connected Providers & Services

We work with trusted modular integration libraries (including Google AdMob trackers and operational crash logs) to optimize rendering, track exceptions, and display promotional marketing units. These third-party entities track diagnostic device fingerprints under industry-standard compliance loops.

5. Amendments to This Policy

We may occasionally iterate on our structural privacy details. System updates will be documented instantly across this portal. We recommend routine verification of this document grid to check security postures.
